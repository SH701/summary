# AI Document Summary Lab

AI Document Summary Lab은 PDF 문서를 업로드하면
텍스트를 추출해 AI가 바로 요약해주는 미니 프로젝트입니다.
텍스트가 없는 스캔본 PDF는 자동으로 안내 메시지를 제공합니다.

## 사이트 
https://summary-plum.vercel.app/

## 기능

PDF 업로드: 사용자가 파일만 올리면 서버가 자동 처리

텍스트 추출(pdf-parse): PDF 내부 텍스트를 빠르게 파싱

AI 요약(gpt-4o-mini): 3–5문장으로 자연스러운 한국어 요약 생성

추출 불가 안내: 스캔본/보호된 문서 등 텍스트가 없으면 깔끔하게 메시지 반환



## 제한 사항

이 프로젝트는  아래 기능은 포함하지 않습니다.

1. OCR 기반 이미지 → 텍스트 변환

1. 긴 문서를 chunk 단위로 나누는 단계적 요약

3. 태그/카테고리 자동 분류

4. 표/이미지 등 복잡한 PDF 레이아웃 분석
